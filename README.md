# MTT Teacher Web

> 医疗思维训练教师端管理系统

基于 Vue 2.0 全家桶构建的医疗思维训练教师端管理平台，提供课程管理、学员管理、考试管理、作业管理、病例管理等功能。

## 📋 项目简介

本项目是一个医疗思维训练教师端管理系统，主要用于教师端对课程、学员、考试、作业、病例等内容进行管理和操作。

## 🛠 技术栈

- **框架**: Vue 2.5.21
- **路由**: Vue Router 3.0.2
- **状态管理**: Vuex 3.0.1
- **UI 组件库**: View Design (iView) 4.0.2
- **构建工具**: Webpack 4.47.0
- **HTTP 客户端**: Axios 0.18.0
- **图表库**: ECharts 4.7.0, Chart.js 2.7.3
- **其他**: Socket.io, Moment.js, Lodash 等

## 📦 环境要求

- Node.js >= 4.0.0
- npm >= 3.0.0

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发环境

启动开发服务器（支持热重载，默认端口 4000）

```bash
npm run dev
```

### 生产环境

构建生产版本

```bash
npm run build
```

启动生产服务器

```bash
npm run server
```

### 其他构建命令

```bash
# 构建生产版本（备用）
npm run build2

# 测试环境构建
npm run test

# 测试环境构建（备用）
npm run testxg

# 启动测试服务器
npm run start
```

## 📖 功能模块

### 核心功能

- **用户管理**: 用户查看、编辑、权限分配、学员创建与管理
- **课程管理**: 线上课、线下课的产品创建与管理
- **考试管理**: 考试创建、批阅、分析
- **作业管理**: 作业创建、批阅、管理
- **病例管理**: 病例库管理、病例创建与编辑
- **数据管理**: 数据概览、考试分析、病例分析
- **个人中心**: 个人信息管理

## 🔄 产品流程

### 线上课流程

1. **创建产品** - 产品状态：编辑中
2. **添加讲师**（可选）- 产品状态：编辑中
3. **添加课程资料** - 启用课程资料（可选）- 产品状态：编辑中
4. **添加推荐资料** - 启用推荐资料（可选）- 产品状态：编辑中
5. **添加线上课**（可选）- 产品状态：编辑中
6. **产品信息** - 查看 → 添加 → 课程列表 - 产品状态：编辑中
7. **修改产品状态** - 改为"正常" - 产品状态：正常

### 线下课流程

1. **添加/复制学期**
2. **添加课程**
3. **启用线下课**
4. **发送学期** → **发送线下课**

### 任务包流程

1. **添加分类**
2. **创建任务**
3. **激活任务** - 手动激活/定时激活

### 权限管理流程

1. **查看用户** → **编辑** → **分配权限**
2. **创建学员**
3. **查看学员** → **查看详情** → **线上课** → **解锁测验**
